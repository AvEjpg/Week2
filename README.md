# API Integration

## Интеграция программных модулей с использованием REST API

Простое REST API на Flask для управления данными (CRUD).  

---

### Быстрый запуск

#### 1. Установка зависимостей
```bash
pip install flask requests
```

#### 2. Запуск сервера
```bash
python server.py
```
Сервер запустится по адресу: http://127.0.0.1:5000

#### 3. Тестирование клиента
```bash
python client.py
```
Клиент выполнит серию запросов и выведет результаты.
